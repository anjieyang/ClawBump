# Privacy Notes

ClawBump is designed so that durable storage covers control-plane state, not full
chat transcripts.

## Stored by default

- anonymous install profile
- onboarding facets
- match requests and deferred invites
- shadow session membership and consent state
- feedback, reports, and blocks

## Not stored durably by default

- ordinary chat message transcripts
- raw channel identities from the user's underlying messaging provider

## Exception path

User-authorized evidence excerpts may be stored when reporting abuse.
